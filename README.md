# Exam Portal Prototype

## Overview
This project is an exam portal prototype featuring separate user interfaces for **Admins** and **Candidates**. The frontend is built using **React, HTML, and CSS**, while the backend uses **JSON Server** with **Axios** for API communication.

---

## Tech Stack

### Frontend
- React
- HTML5
- CSS3
- JavaScript

### Backend
- JSON Server (mock REST API)

### API & Networking
- Axios

### Development Tools
- Node.js
- npm

---

## Admin Features
- Login page
- Dashboard displaying invited users with options to update or delete users
- Registration page to invite new users
- Success page confirming that invitations have been sent

---

## Candidate Features
- Sign-up page for new users
- Login page for returning users
- Dashboard displaying available exams
- Camera check and recording consent confirmation page
- Examination start page
- Success page confirming exam submission
